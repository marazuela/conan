## Cold-start verification questions

A new session reading this INDEX should be able to answer:
1. Where do I find the 7-dimension rubric? → `framework/scoring_system.md`.
2. What is the current phase? → `SESSION_STATE.md` (currently Phase 0 complete, Phase 1 next).
3. Where are the scheduled-task definitions? → `skills/<task-id>/SKILL.md` per D-013.
4. What are the open questions? → `OPEN_QUESTIONS.md` (Q-001, Q-002, Q-003).
5. Which channels are in scope? → `OBJECTIVES.md` §Scope (six channels).
6. How are parties resolved? → `CONTEXT.md` §Entity Resolution Protocol.
7. How do I know the endpoint is verified? → `CONTEXT.md` endpoint-table Status column (currently all `⚠️ UNVERIFIED`).
